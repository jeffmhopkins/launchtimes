# Launch Timelines
* Live at https://LetsFalconGo.app
* Displays launch times for the three Falcon 9 Launch Pads
* No ads or extra clicks for easy T-Time
* API from https://thespacedevs.com/llapi

# Mobile Install - Android / Apple
* Can be installed as Progressive Web Application for Native like operation
* Android -> Open link in Chrome, click menu, 'Add to Home Screen"
* iOS -> Open link in Safari, click share, "Add to Home Screen"
* Not sure on full compatibility across all devices... this is using some pretty generic javascript, but who knows
